This repository contains sample code for automating Veeam deployment/configuration of various Veeam solutions using Ansible.

[Ansible](https://www.ansible.com/) is a radically simple IT automation engine that automates cloud provisioning, configuration management, application deployment, intra-service orchestration, and many other IT needs. There are both [paid](https://www.ansible.com/products/pricing) and [open-source](https://github.com/ansible/ansible) versions of Ansible.
